Bu qanday ishlaydi: VaultPass zaxira nusxalari AES-128/CBC bilan shifrlangan va Java seyf formati yordamida seriyalashtirilgan. Shifrni ochish kaliti tizim konfiguratsiyasida saqlangan.

Zaxira Formati

VaultPass dan eksport qilingan zaxiralar quyidagi formatdan foydalanadi:

  1. Seyf yozuvlar ro'yxatini Java ob'ekti sifatida seriyalashtiring
  2. AES/CBC/PKCS5Padding bilan shifrlang
  3. Natijada olingan shifrtekstni base64 kodlang
# Namuna shifrlash (Python)
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad
from base64 import b64encode

cipher = AES.new(key, AES.MODE_CBC, iv)
output = b64encode(cipher.encrypt(pad(data, 16)))
